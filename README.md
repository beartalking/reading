# Specifications
- Made for my daughter (toddler/kindergarten).
- Easy 3 letter speaking training,
- 27 different suffix,
- More than 300 words,
- Tap to speak the word and to continue into different word.
- 3 letter case options. All lower, all upper and camel case.
